Confianza - Comunicación - Transparencia

Cliente

Lo mas importante para que el proyecto sea exitoso es que el cliente tenga:

  • Compromiso.
  • Responsabilidad.

Cliente

  • Realizando pruebas.
  • Comunicación diaria (skype).
  • Product Owner dedicado al proyecto.
  • Tener el foco del proyecto definido.

Contratación

  • precio por Sprint.
  • Precio por hora.
  • se escuchan propuestas.

*Primer Sprint GRATIS.

*Primer Sprint

  • Establecer etapas manejables.
  • Establecer reglas de relación.
  • Descubrir valor del producto.
  • Establecer estructura del proyecto.
  • Entregar producto con el mayor valor posible (sprint).

Objetivo

  • Subir productividad.
  • Mayor satisfacción equipo (cliente - proveedor).
  • Reducción de costos.
  • Riesgo compartido.
  • Limite al alcance.
  • No garantizar fecha de entrega.

Tipos de contrato

  • Gruesa: Contrato por año (se afinan precios cada año).
  • Fina: contrato por meses (se renuevan cada dos meses).

Clausulas

Ejemplo:

  • acción: Tiempo de respuesta lento del cliente.
  • reacción: Se aumenta un sprint.
  • acción: No recibir Feedback al finalizar sprint.
  • acción: Se da por cerrado el sprint y se continua con el siguiente.
